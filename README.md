# Simple Python Four Function Calculator
This project is a Python-based calculator application that allows users to perform basic arithmetic operations while keeping a persistent record of calculations. It demonstrates Python fundamentals, list handling, file I/O, and program flow control. The calculator includes features such as viewing past calculations and clearing the history, making it more interactive and user-friendly.

Features:
Basic Operations: Add, Subtract, Multiply, Divide

Calculation History: Automatically saves all calculations to a history.txt file

View History: Displays all previous calculations stored in memory and file

Clear History: Users can clear calculation history from memory and file with confirmation

Error Handling: Handles invalid input and division by zero
Example: 
Simple Calculator
1. Add
2. Subtract
3. Multiply
4. Divide
5. Exit
6. View calculation history
7. Clear calculation history
